👉 **[link.promstack.com](https://link.promstack.com)** 에서 시작하세요.

## 📊 대시보드 한눈에 보기

로그인하면 가장 먼저 보이는 **대시보드**입니다.

### 대시보드에서 확인할 수 있는 것

| 섹션 | 내용 |
|------|------|
| **통계** | 저장된 링크, 폴더, 태그, 하이라이트 수 |
| **최근 활동** | 최근 저장한 링크들 |
| **구독 피드** | RSS 구독에서 새로 온 글 |
| **오늘의 발견** | AI가 추천하는 다시 볼 만한 링크 |
| **내 팀** | 소속된 팀 바로가기 |
| **북마크** | 즐겨찾기한 중요 링크 |

시간대별로 **인사말**도 바뀝니다. 아침엔 "좋은 아침입니다", 저녁엔 "오늘 하루도 수고하셨어요" 🌙

## 🔍 링크 검색하기

저장된 링크가 많아지면 **검색**이 중요합니다.

### 검색 범위

LinkStack은 URL만 검색하는 게 아닙니다:

- ✅ **제목** 검색
- ✅ **URL** 검색  
- ✅ **본문 내용** 검색 (저장 시 추출된 텍스트)
- ✅ **AI 요약** 검색

> 💡 **예시**: "React 상태관리"를 검색하면 제목이나 URL에 없더라도, 본문에 해당 내용이 있는 링크가 모두 검색됩니다.

### 필터링 옵션

| 필터 | 설명 |
|------|------|
| **폴더별** | 특정 폴더의 링크만 보기 |
| **태그별** | 특정 태그가 붙은 링크만 |
| **기간별** | 오늘, 어제, 1주일, 1개월 등 |
| **북마크** | 즐겨찾기한 링크만 |

---

## 📂 폴더로 정리하기

### 폴더 구조

LinkStack은 **계층형 폴더**를 지원합니다:

```
📁 개발
   📁 Frontend
      📁 React
      📁 Vue
   📁 Backend
      📁 Node.js
📁 읽을거리
   📁 기술 블로그
   📁 뉴스
```

### 폴더 만들기

1. 좌측 사이드바 → **폴더 관리** (또는 정리 메뉴)
2. **새 폴더** 클릭
3. 이름 입력, 아이콘 선택 (선택사항)
4. 상위 폴더 선택 (하위 폴더로 만들 경우)

### 링크를 폴더로 이동

- 링크 상세 페이지에서 **폴더 변경**
- 링크 목록에서 **드래그 앤 드롭** (데스크톱)
- 여러 링크 선택 후 **일괄 이동**

---

## 🏷️ 태그 활용하기

폴더는 **하나만** 지정할 수 있지만, 태그는 **여러 개** 붙일 수 있습니다.

### 태그 생성

1. **정리** 메뉴 → **태그** 탭
2. **새 태그** 클릭
3. 이름과 **색상** 선택

### 태그 색상 활용 예시

| 색상 | 용도 |
|------|------|
| 🔴 빨강 | 긴급/중요 |
| 🟡 노랑 | 읽는 중 |
| 🟢 초록 | 완료됨 |
| 🔵 파랑 | 참고자료 |
| 🟣 보라 | 나중에 |

---

## 🤖 AI 요약 활용하기

LinkStack은 저장된 페이지의 내용을 **LLM으로 자동 분석**합니다.

### AI 요약에서 제공하는 정보

| 항목 | 설명 |
|------|------|
| **한줄 요약** | 50자 내외의 핵심 요약 |
| **상세 요약** | 200자 내외의 자세한 요약 |
| **키워드** | 자동 추출된 핵심 키워드 5~10개 |
| **핵심 포인트** | 주요 내용 3~5개 |

### AI 요약 생성하기

1. 링크 상세 페이지 이동
2. **🤖 AI 요약 생성** 버튼 클릭
3. 잠시 후 요약 결과 확인

> ⚡ **팁**: 본문이 200자 이상인 페이지에서만 AI 요약이 생성됩니다.

## 👥 팀 협업 기능

혼자 쓰기 아깝다면? **팀**을 만들어 동료들과 함께 사용하세요.

### 팀 만들기

1. **팀** 메뉴 → **새 팀 만들기**
2. 팀 이름과 설명 입력
3. 완료!

### 팀원 초대

1. 팀 상세 페이지 → **멤버 관리**
2. **멤버 초대** 클릭
3. 이메일 입력 (이미 가입된 사용자)
4. 역할 선택 (관리자 또는 멤버)

### 팀에 링크 공유

저장된 링크를 팀에 공유하는 방법:

1. 링크 상세 페이지에서 **공유** 버튼
2. 공유할 팀 선택
3. 선택적으로 의견 추가
4. **공유** 완료

### 팀에서 할 수 있는 것

- 📚 **공유된 링크** 모아보기
- 💬 **의견 교환** - 링크에 대해 토론
- 📊 **팀 통계** - 누가 얼마나 공유했는지

## 📰 RSS 구독 기능

좋아하는 블로그나 사이트의 새 글을 **자동으로 수집**합니다.

### 구독 추가하기

1. **구독** 메뉴 → **새 구독 추가**
2. RSS 피드 URL 입력 (또는 사이트 URL - 자동 탐지)
3. 저장할 폴더 선택 (선택사항)
4. 업데이트 주기 설정

### 추천 피드 카테고리

LinkStack은 분야별 **인기 피드**를 추천합니다:

| 분야 | 예시 사이트 |
|------|------------|
| 기술 | Hacker News, TechCrunch, The Verge |
| 개발 | DEV.to, Medium Engineering, CSS-Tricks |
| AI | OpenAI Blog, Google AI Blog |
| 비즈니스 | TechCrunch, Bloomberg Tech |
| 디자인 | Dribbble, Awwwards |

### 피드 새 글 확인

- 대시보드에서 **구독 피드 새 글** 섹션 확인
- **구독** 메뉴에서 구독별로 새 글 보기
- 원하는 글 클릭해서 저장 또는 읽기

## ⭐ 북마크 기능

모든 저장된 링크 중 **특히 중요한 것**은 북마크하세요.

### 북마크하기

- 링크 카드의 **⭐ 아이콘** 클릭
- 링크 상세 페이지에서 **북마크** 버튼

### 북마크 모아보기

- 대시보드 사이드바의 **북마크** 섹션
- 링크 목록에서 **북마크 필터** 적용

## ✏️ 하이라이트 관리

확장 프로그램에서 저장한 **하이라이트**를 웹에서 관리합니다.

### 하이라이트 메뉴

1. **하이라이트** 메뉴 → 모든 하이라이트 목록
2. 색상별 필터링
3. 해당 링크로 바로가기
4. 하이라이트 삭제 또는 수정

### 하이라이트 활용법

| 색상 | 용도 예시 |
|------|----------|
| 🟡 노랑 | 핵심 개념 |
| 🟢 초록 | 예시/코드 |
| 🔵 파랑 | 출처/참고 |
| 🟣 분홍 | 의문점/나중에 확인 |
| 🟠 주황 | 인용할 구절 |

## ⚙️ 설정하기

**설정** 메뉴에서 다양한 옵션을 조정하세요:

### 계정 설정

- 프로필 사진, 이름 변경
- 이메일 알림 설정
- 게스트 → 정식 계정 전환

### 테마 설정

- **라이트 모드** ☀️
- **다크 모드** 🌙
- **시스템 설정 따르기** (권장)

### 개인정보 설정

- 기본 공개 범위 (비공개/공개)
- 활동 표시 여부
- 데이터 내보내기/삭제
