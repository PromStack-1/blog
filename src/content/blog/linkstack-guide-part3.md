혼자 링크를 모으는 것도 좋지만, **팀과 함께** 하면 더 강력해집니다. LinkStack의 **팀 협업**과 **커뮤니티** 기능을 알아봅니다.

---

## 👥 팀 기능 소개

팀 기능을 사용하면:

- 📚 **링크 공유** - 유용한 링크를 팀원들과 공유
- 💬 **의견 교환** - 링크에 대해 토론하고 피드백
- 📊 **팀 통계** - 팀의 지식 축적 현황 확인
- 🔔 **알림** - 새 공유와 댓글 알림

---

## 🆕 팀 만들기

### 새 팀 생성

1. 좌측 메뉴 **팀** 클릭
2. **새 팀 만들기** 버튼
3. 정보 입력:
   - **팀 이름**: 팀을 나타내는 이름
   - **설명**: 팀의 목적 (선택사항)
4. **생성** 완료

### 팀 역할

| 역할 | 권한 |
|------|------|
| **소유자** | 모든 권한, 팀 삭제 가능 |
| **관리자** | 멤버 초대/제거, 링크 관리 |
| **멤버** | 링크 공유, 댓글 작성 |

---

## 📧 팀원 초대하기

### 이메일로 초대

1. 팀 상세 페이지 → **멤버** 탭
2. **멤버 초대** 클릭
3. 초대할 사람의 **이메일** 입력
4. **역할** 선택 (관리자 or 멤버)
5. 초대 완료!

> 📌 **참고**: 초대받는 사람이 LinkStack에 이미 가입되어 있어야 합니다.

### 초대 링크로 초대

1. 팀 설정 → **초대 링크 생성**
2. 링크 복사해서 공유
3. 링크를 받은 사람이 클릭하면 자동 가입

---

## 🔗 팀에 링크 공유하기

### 방법 1: 링크 상세에서 공유

1. 공유할 링크의 상세 페이지 이동
2. 우측 상단 **공유** 버튼
3. 공유할 팀 선택
4. 의견 추가 (선택사항)
5. **공유** 완료

### 방법 2: 저장하면서 바로 공유

1. 확장 프로그램으로 링크 저장 시
2. **팀에 공유** 옵션 켜기
3. 팀 선택
4. 저장과 동시에 공유!

### 공유할 때 팁

| 좋은 습관 | 이유 |
|----------|------|
| **간단한 의견 추가** | 왜 공유하는지 맥락 제공 |
| **관련 텍스트 인용** | 핵심 내용 강조 |
| **적절한 팀 선택** | 관련된 팀에만 공유 |

---

## 💬 팀에서 토론하기

### 댓글 달기

1. 팀 피드에서 공유된 링크 클릭
2. 하단 **댓글** 영역
3. 의견 작성 후 **전송**

### 대댓글 (답글)

- 댓글 아래 **답글** 클릭
- 특정 댓글에 대한 답변 작성

### 좋아요 👍

- 유용한 공유나 댓글에 **좋아요** 표시
- 팀 내 인기 있는 링크 확인 가능

### 인용 기능

> "이 부분이 특히 중요해요"

텍스트를 선택하고 **인용** 버튼을 누르면, 해당 내용을 인용하면서 댓글을 달 수 있습니다.

---

## 📊 팀 통계 확인

팀 대시보드에서 다양한 통계를 확인할 수 있습니다:

| 통계 | 내용 |
|------|------|
| **총 공유 링크 수** | 팀에 공유된 전체 링크 |
| **멤버별 기여도** | 누가 얼마나 공유했는지 |
| **인기 링크** | 좋아요/댓글이 많은 링크 |
| **최근 활동** | 최근 공유 및 댓글 |
| **자주 공유되는 도메인** | 팀이 관심 있는 사이트 |

## 🌐 커뮤니티 피드

팀 외에도 **전체 사용자**와 소통할 수 있습니다.

### 커뮤니티 피드란?

- **공개**로 설정된 의견들이 모이는 곳
- 다양한 분야의 흥미로운 링크 발견
- 다른 사용자와 토론

### 피드에 참여하기

1. **피드** 메뉴 클릭
2. 최신/인기/팔로잉 탭 선택
3. 흥미로운 글에 **좋아요** 또는 **댓글**
4. 유익한 사용자 **팔로우**

### 내 의견 공개하기

링크 저장 시 또는 의견 작성 시:

- **공개** 옵션을 켜면 피드에 표시
- **비공개** (기본값)는 나만 볼 수 있음

## 🔔 알림 설정

팀 활동에 대한 알림을 설정하세요:

### 알림 받기

- **새 링크 공유**: 팀에 새 링크가 공유되면
- **내 글에 댓글**: 내가 공유한 글에 댓글이 달리면
- **멘션**: 누군가 나를 태그하면
- **팀 초대**: 새 팀에 초대되면

### 알림 받는 방법

- **웹 알림**: 대시보드에서 🔔 아이콘
- **이메일**: 설정에서 활성화

## 🛡️ 팀 관리 팁

### 좋은 팀 운영을 위한 조언

| 팁 | 이유 |
|----|------|
| **명확한 팀 목적** | 관련 없는 링크 공유 방지 |
| **적절한 멤버 수** | 너무 많으면 잡음, 적으면 비활성 |
| **정기적 정리** | 오래된/관련 없는 링크 아카이브 |
| **인정 문화** | 좋은 공유에 좋아요와 댓글 |

### 문제 상황 대처

- **스팸 링크**: 관리자가 삭제 가능
- **비활성 멤버**: 관리자가 제거 가능
- **팀 삭제**: 소유자만 가능, 데이터는 30일 보관
