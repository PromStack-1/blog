인터넷에서 좋은 글을 발견했을 때, 어떻게 하시나요? 브라우저 북마크에 추가하면 나중에 찾기 어렵고, 메모장에 복사하면 맥락이 사라집니다. **LinkStack**은 이 문제를 해결합니다.

이 가이드에서는 LinkStack 브라우저 확장 프로그램을 설치하고, 모든 기능을 100% 활용하는 방법을 알려드립니다.

---

## 🚀 설치하기

### Chrome 사용자

1. [Chrome 웹 스토어](https://chromewebstore.google.com/detail/linkstack/odjokjnphphbanfiknnniokaigcblehf)에서 **Chrome에 추가** 클릭
2. 설치 완료 후 우측 상단 퍼즐 아이콘 🧩 클릭
3. LinkStack 옆 📌 핀 아이콘 클릭해서 고정

### Whale 사용자

1. [Whale 스토어](https://store.whale.naver.com/detail/nfeimhfehpmkkmfhnjhlbdlipbbodcan)에서 **설치** 클릭
2. 동일하게 확장 프로그램 고정

> � **팁**: 처음에는 로그인 없이 **게스트 모드**로 바로 사용할 수 있어요. 나중에 Google이나 Kakao로 로그인하면 모든 데이터가 자동으로 연동됩니다.

---

## 📌 링크 저장하기

### 방법 1: 키보드 단축키 (가장 빠름!)

저장하고 싶은 페이지에서:

| 운영체제 | 단축키 |
|---------|--------|
| Windows/Linux | `Ctrl + Shift + S` |
| Mac | `⌘ + Shift + S` |

**끝!** 1초만에 현재 페이지가 저장됩니다.

### 방법 2: 우클릭 메뉴

1. 페이지 아무 곳에서 **우클릭**
2. **🔗 LinkStack에 저장** 선택

### 방법 3: 확장 프로그램 아이콘

1. 브라우저 우측 상단의 LinkStack 아이콘 클릭
2. **사이드 패널**이 열리면서 현재 페이지 정보 표시
3. 폴더와 태그 선택 후 저장

---

## ✨ LinkStack이 자동으로 수집하는 것들

단순히 URL만 저장하는 게 아닙니다. LinkStack은 페이지에서 다음 정보를 **자동으로 추출**합니다:

| 수집 항목 | 설명 |
|----------|------|
| 📄 **본문 내용** | 페이지의 핵심 텍스트 (최대 5,000자) |
| 🖼️ **대표 이미지** | OG 이미지, 본문 내 주요 이미지 |
| 🏷️ **메타데이터** | 작성자, 게시일, 설명 |
| 🔗 **관련 링크** | 본문에 포함된 외부 링크들 |
| 📑 **목차** | h1, h2, h3 헤딩 구조 |
| 🔑 **키워드** | 자동 추출된 핵심 키워드 |

이 모든 정보가 함께 저장되어 나중에 **전체 검색**이 가능합니다!

## ✏️ 하이라이트 기능

중요한 문장을 형광펜으로 표시하듯 **하이라이트**할 수 있습니다.

### 하이라이트 추가하기

1. 웹 페이지에서 텍스트를 **드래그해서 선택**
2. 나타나는 팝업에서 **✏️ 하이라이트** 클릭
3. 원하는 색상 선택 (노랑, 초록, 파랑, 분홍, 주황)
4. 선택적으로 메모 추가

### 하이라이트 확인하기

- **웹 대시보드 → 하이라이트** 메뉴에서 모든 하이라이트 확인
- 해당 페이지 재방문 시 하이라이트가 **자동으로 표시**됨

> 📚 **활용 예시**: 기술 문서를 읽으면서 중요한 개념을 하이라이트하고, 나중에 복습할 때 하이라이트만 모아보기

## 💬 의견 공유 기능

LinkStack은 단순한 북마크 도구가 아닙니다. **링크 기반 SNS** 기능도 있어요!

### 의견 남기기

| 단축키 | 설명 |
|--------|------|
| `Ctrl + Shift + E` (Win) | 현재 페이지에 의견 추가 |
| `⌘ + Shift + E` (Mac) | 현재 페이지에 의견 추가 |

또는 우클릭 → **💬 의견 추가**

### 인용하면서 의견 남기기

1. 페이지에서 텍스트 선택
2. 우클릭 → **📝 선택한 텍스트와 함께 저장**
3. 해당 텍스트가 인용된 상태로 의견 작성

### 공개 피드

- 공개로 설정된 의견은 **커뮤니티 피드**에 표시
- 다른 사용자의 의견에 **좋아요, 댓글** 가능
- 흥미로운 링크 발견의 새로운 방법!

## 📂 폴더와 태그로 정리하기

### 저장하면서 바로 정리

링크 저장 시 **폴더**와 **태그**를 선택할 수 있습니다:

1. 확장 프로그램 사이드 패널 열기
2. **폴더 선택** (없으면 새로 생성)
3. **태그 추가** (색상 지정 가능)
4. 저장 완료!

### 추천 정리법

| 폴더 구조 예시 | 용도 |
|---------------|------|
| 📁 개발/React | 기술 문서 |
| 📁 개발/TypeScript | 기술 문서 |
| 📁 읽을거리/기술 | 나중에 읽을 기사 |
| 📁 읽을거리/뉴스 | 뉴스 기사 |
| 📁 업무/프로젝트A | 특정 프로젝트 관련 |

> 💡 **팁**: 폴더는 **큰 카테고리**, 태그는 **세부 분류**로 사용하세요. 예: 폴더 "개발" + 태그 "React", "상태관리"

## � 사이드 패널 활용하기

확장 프로그램 아이콘을 클릭하면 **사이드 패널**이 열립니다.

### 사이드 패널에서 할 수 있는 것

| 기능 | 설명 |
|------|------|
| **현재 페이지 저장** | 폴더/태그 선택하며 저장 |
| **커뮤니티 피드** | 다른 사용자들의 공개 링크 |
| **빠른 검색** | 저장된 링크 검색 |
| **최근 저장** | 최근 저장한 링크들 |

사이드 패널을 **열어둔 채로 웹서핑**하면, 계속 링크를 저장하고 피드를 확인할 수 있어요.

## ⌨️ 단축키 모음

자주 사용하는 단축키를 정리했습니다:

| 기능 | Windows/Linux | Mac |
|------|--------------|-----|
| **현재 페이지 저장** | `Ctrl+Shift+S` | `⌘+Shift+S` |
| **의견 추가** | `Ctrl+Shift+E` | `⌘+Shift+E` |

### 단축키가 안 되나요?

다른 확장 프로그램과 충돌할 수 있습니다. 설정 방법:

1. 브라우저 주소창에 `chrome://extensions/shortcuts` 입력
2. LinkStack 찾아서 원하는 단축키로 변경

## � 게스트 모드에서 정식 계정으로

처음에 게스트로 사용하다가 로그인하면?

1. 기존 게스트 데이터가 **모두 유지**됩니다
2. 자동으로 정식 계정과 **병합**
3. 다른 기기에서도 **동기화** 가능

> 🔒 **보안**: 게스트 데이터는 기기에만 저장됩니다. 로그인하면 서버에 안전하게 백업됩니다.
